- 模型：作为核心推理引擎，处理输入输出，执行逻辑判断。
- 工具：扩展模型能力，使 Agent 能够读取本地文件、执行计算等操作。
- 记忆：存储全局知识和历史对话上下文，为 Agent 提供持续的信息基础。
- 执行循环：模型、工具与记忆协同工作的循环流程，驱动 Agent 完成任务。

2. 工作机制
- 工具调用：模型通过工具调用扩展自身能力，将结果用于后续回答。
- 记忆管理：Memory 模块为 Agent 提供全局知识和历史对话上下文，确保对话连贯性。

Agent 系统是一个由模型、工具、记忆和执行循环组成的智能体框架，通过协同工作实现自动化任务处理。
